# Google Workspace Referral Integration v4.3.1

確認日：2026-07-20

## 反映内容

- Google Workspaceご紹介プログラム専用リンクを中央設定へ登録
- 日本語の購入意図が強い6記事へ紹介CTAを反映
- 広告・紹介関係の開示と `rel="sponsored nofollow noopener"` を自動出力
- 初年度割引コードの問い合わせをLuQvia公式LINEへ誘導
- 紹介プログラム管理状態を `active-referral` へ更新
- 紹介リンク：https://referworkspace.app.goo.gl/ewLM

## プロモーションコードの管理方針

Google Workspaceのプロモーションコードは1コードにつき1件の紹介先のみ利用できます。Google公式の案内に従い、コードはWebページ、公開ドキュメント、GitHubリポジトリ、公開ZIPへ保存しません。紹介先へ個別に配布し、使用済み・割当済みの状態は別の非公開台帳で管理します。

## 承認済みと誤認させない表現

Google Workspaceご紹介プログラム参加者として紹介リンクを掲載しますが、「Google公式パートナー」「Google認定代理店」等の表現は使用しません。
