# Luqevora v3.7.0 — Integrated Revenue Edition

This release merges the v3.6.0 Store DX branch with the uploaded v3.4.0 XServer affiliate branch.

## Merge policy

The v3.6.0 Store DX project was used as the baseline because it contains the latest restaurant POS, mobile-order, funfo, Kantan Chumon, and EMEAO! content. The uploaded XServer branch was then applied through its reproducible update script rather than copying generated HTML over the newer project.

This preserves:

- Store DX navigation, category pages, products, articles, affiliate materials, and submission documents.
- The full XServer and XServer Business article cluster and supplied A8.net materials.
- Existing static-generation, product-database, SEO-image, revenue-report, operations-report, sitemap, RSS, hreflang, and validation workflows.

## Homepage balance

Featured content now includes both Store DX and hosting comparisons and reviews. This prevents the XServer branch from replacing the newer Store DX homepage configuration.

## Quality upgrades

The first Store DX articles were strengthened with additional official references and expanded English implementation guidance. Both the content audit and public validation complete with zero errors and zero warnings.
