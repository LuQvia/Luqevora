# Luqevora v2.0 Phase 1 QA

## Scope
- Shared editorial scorecard for review articles
- Shared best-fit / caution panel for every generated article
- Goal-based entry cards on the home page
- Category content-mix dashboard and curated start-here section
- Unified v5.0 presentation styles
- Existing article JSON, affiliate materials, canonical URLs, and slugs retained

## Editorial safety
The scorecard is explicitly labeled as a Luqevora editorial assessment. It does not claim to be an aggregate of customer reviews or measured product performance. Scores are generated consistently from content completeness and category-oriented editorial criteria.

## Release validation
Run `npm run check`. The release is acceptable only when the final validator reports zero errors and zero warnings.
