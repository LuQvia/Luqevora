# QA v3.2.0 Operations Dashboard

## Scope
- Internal operations dashboard generated under `reports/`
- Article and product verification schedules
- Priority queue by content type, affiliate use, featured status and source coverage
- Affiliate-key configuration audit
- Official-source URL syntax and reuse audit
- Machine-readable JSON report

## Safety and publication
The dashboard is not copied to `public/`. It contains editorial operations data and remains an internal release artifact.

## Review cadence
- Review: 30 days
- Comparison: 45 days
- Guide: 90 days
- Product catalog: 45 days

## Commands
- `npm run operations`
- `npm run check`
