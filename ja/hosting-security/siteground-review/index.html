結論：日次バックアップとWordPress運用機能を全体でまとめやすいが、更新価格の上昇を織り込む必要がある

判断の要点

SiteGround

SiteGroundはWebホスティングとWordPress管理機能をSite Toolsで提供します。全プランにSSL、CDN、日次バックアップ、メール、移行、24時間サポートを案内しています。初回価格は12か月前払いのキャンペーンで、更新時の価格差を必ず総額へ反映します。

公式情報から確認できる利点

契約前に確認する制約

  • 初回価格と更新価格の差が大きい
  • 表示価格はVAT別で12か月前払い
  • プランごとにサイト数・容量・高度機能が異なる
  • 「unlimited traffic」でも実際のリソース条件は確認が必要

日次バックアップは30日履歴、上位プランはオンデマンド作成

自動バックアップ

SiteGroundはサイト全体を毎日自動バックアップし、30日分の履歴を保持すると案内しています。復元方法と対象範囲を契約後に確認します。

GrowBig・GoGeek

上位2プランはオンデマンドバックアップを含みます。GoGeekはステージングに加えGitも案内されています。

費用上の注意：初回価格ではなく、更新価格とVATを含む2年目以降の総額で比較してください。

向いている事業者・別の選択肢も比較すべき事業者

向いている可能性がある

  • 日次バックアップ、SSL、CDN、メールをまとめたい
  • WordPressの自動更新・移行・ステージングを使いたい
  • 複数サイトをGrowBig以上で管理したい
  • 技術担当者と共同作業者のアクセスを分けたい

別の選択肢も比較

  • 初回から更新まで料金を一定にしたい
  • 最安の長期ホスティングだけを求める
  • root権限や独自サーバー構成が必要
  • WordPress専業の訪問数課金型サービスを求める

よくある質問

StartUpの2.99米ドルは更新後も続きますか？

いいえ。公式ページでは12か月前払いの初回価格で、更新は月17.99米ドル、VAT別と案内されています。

GrowBigを選ぶ主な理由は何ですか？

確認時点では複数サイト、50GB、オンデマンドバックアップ、ステージングがStartUpとの差です。

バックアップは何日保存されますか？

公式ページは毎日自動バックアップを作成し、30日分の履歴を保持すると案内しています。
