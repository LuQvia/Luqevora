Cloudways Review: Pricing, Hourly Billing, Backups and Email Explained

Cloudways Flexible reviewed through official pricing, hourly billing, backups, bandwidth, email and server-management documentation.

Published and checked: 2026-07-16Estimated reading time: 11 minBy Luqevora Editorial Team

Research method: We checked Cloudways' official pricing, billing, backup, bandwidth, email and server-management documentation on July 16, 2026. Prices, infrastructure options, regions, tax and add-ons can change, so confirm the final configuration in the Cloudways console.

Review scope: This page focuses on Cloudways Flexible. We did not run independent speed tests, load tests or support-quality scoring.

Affiliate disclosure: This page contains no affiliate links at publication. Any future compensated link will be disclosed next to the link and under our Affiliate Disclosure.

Verdict: a managed cloud option for businesses that want infrastructure choice and usage-based billing

Decision summary

Cloudways Flexible

Cloudways adds a managed layer to cloud infrastructure such as DigitalOcean and lets teams operate WordPress and PHP applications. Official pricing lists a 1GB DigitalOcean server from $11 per month or $0.0153 per hour. It offers more infrastructure control than shared hosting, but backup storage, bandwidth overages, email services and the absence of root access must be planned.

Officially documented advantages

  • Choose the infrastructure provider and server size
  • Free Let's Encrypt SSL, staging, SSH/SFTP and monitoring
  • Backup frequency from one hour to seven days and retention from one to four weeks
  • Multiple applications can run on one server

Contract checks and limitations

  • Business mailboxes and transactional email require separate configuration

Backups and data portability

Default configuration

Cloudways documents a default backup every 24 hours with one-week retention for new servers. Frequency can be set from one hour to seven days, and retention from one to four weeks.

On-demand and download

Server-level and application-level on-demand backups are available. Enabling local backups allows the latest backup to be downloaded through SSH or SFTP.

Cost note: off-site backup is a required separately billed feature. The official pricing page lists $0.033 per GB.

Who may benefit and who should compare alternatives

Potential fit

  • Teams running multiple WordPress or PHP applications on one server
  • Businesses that want to scale cloud resources gradually
  • Short-lived testing environments that benefit from hourly billing
  • Agencies and developers using SSH/SFTP and staging

Compare alternatives

  • You want email, domains and a site builder in one plan
  • You do not want to manage resource or bandwidth choices
  • Root access is mandatory
  • You need a completely fixed price including backup storage

Frequently asked questions

What is Cloudways' lowest listed price?

The official pricing page lists a 1GB DigitalOcean server from $11 per month or $0.0153 per hour. Tax, backups, bandwidth overages and add-ons require separate confirmation.

Are backups free?

Backup functionality is built in, but off-site backup storage is billed separately. The official price is listed as $0.033 per GB.

Does the plan include domain email?

Cloudways does not bundle a conventional mailbox service into the server price. It documents Rackspace Email and external SMTP options for mailboxes and transactional email.
