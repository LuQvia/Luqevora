結論：法人メールと複数サイトの運用体制を重視する企業に向く

CPIは、個人ブログをできるだけ安く始めたい利用者より、企業サイト、法人メール、複数ドメイン、制作会社の更新作業を一つの契約で整理したい組織に向く共用レンタルサーバーです。主契約ドメインのWeb容量はSSD 300GB、マルチドメインは1契約につき10件までで、各ドメインに独立したサーバー領域が用意されます。

判断の中心は、月額の安さではなく、SmartReleaseによるテスト環境と本番反映、Web・メール・コントロールパネルを分離した構成、SPF・DKIM・DMARC対応、SLAの範囲です。標準機能と有料オプションの境界を確認してから契約します。

  • 向いている：企業サイトとメールを同じ事業者で管理したい
  • 向いている：複数の顧客サイトや事業サイトを分離して運用したい
  • 別候補も比較：低価格、容量、WordPress移行代行を最優先する

料金と契約期間

公式料金ページでは3か月、6か月、12か月の契約期間が案内されています。12か月契約は月額換算4,840円、合計58,080円で、初期費用は無料です。6か月契約は月額換算5,170円、3か月契約は月額換算5,500円で、短期契約には初期費用22,000円が設定されています。表示価格は税込みで、契約期間分を一括で支払います。

キャンペーンや料金改定があり得るため、記事の金額だけで決めず、申込画面で契約期間、初期費用、ドメイン費用、オプション費用を再確認してください。

Webサイト運用とSmartRelease

ビジネス スタンダードでは、主契約ドメインにSSD 300GB、マルチドメインごとにSSD 100GBのWeb領域が案内されています。マルチドメインは最大10件で、ドメインごとに専用領域を用意する方式です。

SmartReleaseは、テスト環境、自動バックアップ、本番サイトへの反映、復旧をまとめた制作支援機能です。公開前の確認工程を標準化しやすいため、更新担当者が複数いる企業や制作会社に適しています。

SmartReleaseの標準バックアップはWeb領域とデータベースの合計10GB、最大30世代です。Web容量300GB全体が30世代保存されるという意味ではありません。

メール、セキュリティ、SLA

メールはSPF、DKIM、DMARCに対応し、Webメール、転送、自動応答、メーリングリストなどを利用できます。Webサーバー、メールサーバー、コントロールパネルを分離して運用する構成も公式仕様に記載されています。

WAFとCPI SSLサーバー証明書は標準提供ですが、Web改ざん検知は有料オプションです。SLAは100%を基準としますが、対象は主契約のWebサーバーに限定され、メールサーバーとマルチドメインを搭載したサーバーは対象外です。

サポートと導入時の注意点

標準サポートはメールです。24時間365日の電話・メールサポートは有料オプションであり、標準料金に含まれる24時間電話対応ではありません。サーバー移転代行も一部有料のため、移行対象、作業範囲、DNS切り替え、動作確認の担当を事前に確認します。

公式サイトでは10日間の無料お試しと、入金後の20日間返金保証が案内されています。返金対象外となる費用や申請条件があるため、契約前に最新のキャンセルポリシーを確認してください。

  • 必要なドメイン数とメールアカウント容量を見積もる
  • 標準バックアップ10GBを超える場合は外部保管も設計する
  • SLAの対象範囲と返金申請手順を確認する
  • 24時間サポート、改ざん検知、移転代行の追加費用を確認する

CPIが向くケースと向かないケース

CPIは、法人メール、複数ドメイン、テスト環境、復旧手順を重視する企業に適しています。特に、顧客サイトの更新を行う制作会社や、Web担当者が安全に公開作業を行いたい組織ではSmartReleaseが判断材料になります。

一方、1サイトだけの個人ブログ、月額を最小化したいケース、root権限が必要なシステム、標準で大容量の世代バックアップを求めるケースでは、一般向け共用サーバー、VPS、マネージド専用サーバーも比較してください。

よくある質問

CPIの12か月契約はいくらですか？

公式確認時点では月額換算4,840円、12か月分58,080円で、初期費用は無料です。価格や条件は変更されるため、申込画面で再確認してください。

CPIは24時間365日の電話サポートが標準ですか？

標準はメールサポートです。24時間365日の電話・メール対応は有料オプションとして案内されています。

最大30世代バックアップは300GB全体が対象ですか？

公式仕様ではSmartReleaseの自動バックアップはWeb領域とデータベースの合計10GB、最大30世代です。必要容量が大きい場合は外部バックアップを含めて設計してください。

CPIのSLAはメールサーバーも対象ですか？

SLA対象は主契約のWebサーバーです。メールサーバーとマルチドメインを搭載したサーバーは対象外とされています。
