# Luqevora Website Complete v1.1

GitHub Pagesへそのままアップロードできる静的サイト一式です。

## 公開手順
1. このZIPを解凍します。
2. GitHubの `luqevora-site` リポジトリを開きます。
3. `Add file` → `Upload files` を選択します。
4. **ZIPファイル自体ではなく、解凍した中身をすべて**ドラッグ＆ドロップします。
5. `Commit changes` を押します。
6. `Settings` → `Pages` → `Deploy from a branch` → `main / (root)` を選びます。
7. `Custom domain` が `luqevora.com` になっていることを確認します。
8. DNS設定後、`Enforce HTTPS` を有効にします。

## 計測タグの設定
Luqevora専用のIDを作成した後、`assets/js/analytics-config.js` に入力します。

```js
window.LUQEVORA_ANALYTICS_CONFIG = Object.freeze({
  gtmId: "GTM-XXXXXXX",
  ga4Id: "",
  clarityId: ""
});
```

GTMでGA4を管理する場合、`ga4Id`は空欄のままにして二重計測を避けてください。
